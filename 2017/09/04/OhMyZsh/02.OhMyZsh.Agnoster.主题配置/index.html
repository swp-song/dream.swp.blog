  • 使用 Git Clone下来字体库仓库，进入仓库根目录，执行 install.sh 安装。

修改终端 / iTerm2 字体

系统自带终端

  • 进入：终端 > 偏好设置 > 字体修改为 ：Meslo LG S DZ Regular for Powerline。

修改 iTerm2

  • 进入：iTerm2 > Profiles > Text > Font，修改字体为：Meslo LG S DZ Regular for Powerline。
